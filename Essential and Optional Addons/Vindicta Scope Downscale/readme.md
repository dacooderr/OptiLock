
### What The Mod Does
Vindicta's scope in vanilla is, by default, a 4096x4096 texture that does NOT get mipmapped. This is an issue that will cause framedrops on systems with limited vram. This mod downscales the texture to 1080p.

### Installation
Place this file in your addons folder in Deadlock\game\citadel, just like you would any other mod.
Make sure you have this line in your gameinfo.ini file, inside the searchpaths bracket: ``Game    citadel/addons``
