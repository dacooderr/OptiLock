### What The Mod Does
Reduces the poly count of the soul orb by replacing the model.
### Credits  
This mod was originally created by Jayie. The original upload can be found [here](https://gamebanana.com/mods/600780) on Gamebanana.

### Installation
Place this file in your addons folder in Deadlock\game\citadel, just like you would any other mod.
Make sure you have this line in your gameinfo.ini file, inside the searchpaths bracket: ``Game    citadel/addons``
